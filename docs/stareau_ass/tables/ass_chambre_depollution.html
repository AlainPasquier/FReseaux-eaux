Description

Une installation ou une structure conçue pour traiter ou réduire la charge polluante des eaux usées ou des effluents avant leur rejet dans l’environnement. Elle est généralement intégrée à un système d’assainissement pour améliorer la qualité des eaux avant qu’elles ne soient rejetées dans les cours d’eau ou les réseaux de collecte.
